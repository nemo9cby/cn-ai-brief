# Full translations

Put full English translations here only when the content is safe to republish.

Required JSON shape:

```json
{
  "id": "example-id",
  "title": "Example title",
  "source": "Nemo / licensed source / public domain source",
  "original_url": "https://example.com/original",
  "rights": "owned",
  "translation_text": "Full English translation..."
}
```

Allowed public `rights` values:

- `owned`
- `licensed`
- `permissioned`
- `public_domain`

Anything else will not be rendered as a public full-translation page.
